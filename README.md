# 🦁 Safari Sight Words

A fun, animal-themed sight word game for kindergarteners, built for Jack!

## Features
- 150+ NC SCOS sight words across 3 difficulty tiers
- Voice recognition — say the word out loud!
- Animated animal mascots and rewards
- Stars and gems for motivation
- Smart retry system (miss a word, get a similar one)
- Progress saved between sessions (localStorage)
- Parent report with phonics analysis
- Works as a home screen app on iPhone/iPad

## Local Development

```bash
npm install
npm start
```

Then open http://localhost:3000
